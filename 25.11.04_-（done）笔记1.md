假设此时的信号光与选择的本征光信号为最简单情况下的单色光，其电场分量各自表示如下 ：

信号光：

$$e_s(t) = A_s \cos \omega_s t$$

本振光：

$$e_{lo}(t) = A_{lo} \cos \omega_{lo} t$$

当这两束光在耦合器束合后：

$$e(t) = A_s \cos \omega_s t + A_{lo} \cos \omega_{lo} t$$

并入射到 InGaAs 探测器表面，探测器的输出电流表示为：

$$i(t) = \alpha e^2(t) = \alpha (A_s \cos \omega_s t + A_{lo} \cos \omega_{lo} t)^2$$

经过简化处理后变为：

$$i(t) = \alpha (A_s^2 \frac{1+\cos 2\omega_s t}{2} + A_{lo}^2 \frac{1+\cos 2\omega_{lo} t}{2}) + \alpha A_s A_{lo} (\cos(\omega_s t + \omega_{lo} t) + \cos(\omega_s t - \omega_{lo} t))$$

式中 $e_s$ 和 $e_{lo}$ 分别表示太阳光和 LO 本征光的辐射相关电场的组成部分，$A_s$ 和 $A_{lo}$ 分别表示各自场的振幅，$\omega_s$ 是激光频率，$\omega_{lo}$ 是太阳光频率，$t$ 是在电场中的时间，$i$ 是由快速光电探测器测量的电流。

由于式中 $\cos 2\omega_s t$、$\cos 2\omega_{lo} t$、$\cos(\omega_s t + \omega_{lo} t)$ 属于高频率项，远大于探测器的响应带宽，因而 5.5 式可以简化为：

$$i(t) = \frac{\alpha (A_s^2 + A_{lo}^2)}{2} + \alpha A_s A_{lo} \cos(\omega_s t - \omega_{lo} t)$$

其中第一项为直流 DC 信号，第二项为交流 AC 信号。

直流 DC 信号：

$$i_{dc} = \frac{\alpha (A_s^2 + A_{lo}^2)}{2}$$

此式表示为此探测器对信号光（太阳光）和本振光（激光）的直流响应。

交流 AC 信号：

$$i_{ac} = \alpha A_s A_{lo} \cos((\omega_s - \omega_{lo})t)$$

交流信号经射频放大器及带通滤波器后由肖特二极管转化为 DC 信号，表示为：

$$I_m(t) \propto \beta A_s^2 A_{lo}^2 = \beta I_s I_{lo}$$

因此，外差信号探测的是太阳光强与本振光强的乘积，即激光外差是一个光放大的技术。

由于外差使用的光源为宽带太阳光，则总探测信号可以表示为：

$$I_m^{\alpha}(t) = \beta I_{lo}(t) \int_{v-\Delta v}^{v+\Delta v} I_s(v) dv$$

其中，$\Delta v$ 为射频检测带宽，$v(t)$ 是本振光在时间 $t$ 时的频率。当阳光穿过大气层时，地球上太阳光的光谱功率密度可以表示为：

$$I_s(v) = I \exp[-\alpha(v)]$$

其中 $I$ 假定为一个微小波长范围内的常数，是没有吸收的太阳光的光谱功率密度。而 $\alpha(v)$ 是沿太阳辐射路径的总吸收度。将式 5.11 带入式 5.10：

$$I_m^{\alpha}(t) = \beta I_{lo}(t) I \int_{v-\Delta v}^{v+\Delta v} \exp[-\alpha(v)] dv$$

本工作中使用的是逐点扫描模式，使激光器的工作电流逐点变化，同时用锁相放大器输出频率为 $f$ 的高频正弦波对激光器的注入电流进行调制，激光器的输出频率 $v(t)$ 和外差信号光强 $I_0(t)$ 表示为：

$$v(t) = v_0(t) + a \cos(2\pi ft)$$

$$I_0(t) = \bar{I_0} [1 + i_0 \cos(\omega t + \psi_1) + i_2 \cos(2\omega t + \psi_2)]$$

其中，$v_0$ 为所选激光器的实时工作波数 (单位 $\text{cm}^{-1}$)，$\bar{I_0}$ 为外差信号没有吸收时的平均光强，$a$ 为其调制振幅 (单位 $\text{cm}^{-1}$)，$i_0$ 是外差信号调制的相对强度，$\psi_1$ 是线性的 FM (频率调制) / IM (外差信号强度调制) 的相移。$\psi_2$ 是非线性的 FM/IM 的相移。

当激光外差信号到达探测器，此时透射光信号表示如下：

$$I(t) = I_0(t) \tau(v + a \cos \omega t)$$

当满足 $\alpha(v) \ll 0.1$ 弱吸收条件，利用傅里叶余弦级数展开：

$$\tau(\bar{v} + a \cos \omega t) \approx 1 - \alpha(\bar{v} + a \cos \omega t) = - \sum_{k=0}^{\infty} H_k(\bar{v}, a) \cos(k \omega t)$$

当 $\alpha(v) = PX_{gas} S_i(T) \phi L$，得到 K 阶傅里叶系数

$$H_0(\bar{v}, a) = - \frac{PSXL}{\pi} \int_{-\pi}^{\pi} \phi(\bar{v} + a \cos \theta) d\theta$$

$$H_k(\bar{v}, a) = - \frac{PSXL}{\pi} \int_{-\pi}^{\pi} \phi(\bar{v} + a \cos \theta) \cos k\theta d\theta, \ k = 1, 2, 3...$$

由于在波长调制技术中，$2f$ 信号具有波形对称、峰值位于吸收线、中心信号最强等优点，因此在实际工作中，利用锁相放大器对于外差信号进行解调，以此获得 $X_{2f}$ 和 $Y_{2f}$ 分量。

$$X_{2f} = \frac{G \bar{I_0}}{2} [H_2 + \frac{i_0}{2}(H_1+H_3)\cos\psi_1 + i_2(1+H_0 + \frac{H_4}{2})\cos\psi_2]$$

$$Y_{2f} = -\frac{G \bar{I_0}}{2} [i_0(H_1-H_3)\sin\psi_1 + i_2(1+H_0 - \frac{H_4}{2})\sin\psi_2]$$

无吸收时：

$$X_{2f}^0 = \frac{G \bar{I_0}}{2} i_2 \cos\psi_2$$

$$Y_{2f}^0 = \frac{G \bar{I_0}}{2} i_2 \sin\psi_2$$

去除无吸收，此时二次谐波信号为

$$S_{2f} = \frac{G \bar{I_0}}{2} \left\{ \begin{array}{l} [H_2 + \frac{i_0}{2}(H_1+H_3)\cos\psi_1 + i_2(H_0 + \frac{H_4}{2})\cos\psi_2]^2 \\ + [i_0(H_1-H_3)\sin\psi_1 + i_2(H_0 - \frac{H_4}{2})\sin\psi_2]^2 \end{array} \right\}$$
