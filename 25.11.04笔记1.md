下面给出**以大气光谱学/遥感**为出发点的划分与说明。为兼顾光谱学与遥感工程习惯，我同时写出波长与对应的波数区间（$,\tilde\nu=1/\lambda,$，单位 $\mathrm{cm^{-1}}$）。不同学科边界略有差异，以下取**光谱学主流**定义，并在关键处标注与遥感“窗口”的关系。

# 紫外（UV）

**总范围**：$100$–$400\ \mathrm{nm}$（$\tilde\nu\approx 10^5$–$2.5\times 10^4\ \mathrm{cm^{-1}}$）
 **细分（按吸收机理/大气层吸收）：**

- **EUV/VUV**：$100$–$200\ \mathrm{nm}$（$10^5$–$5\times 10^4$）
   主要被 $\mathrm{O_2}$、$\mathrm{N_2}$ 在高层大气（热层/中间层）完全吸收；包含 $\mathrm{O_2}$ **Schumann–Runge** 带（$\sim 175$–$205\ \mathrm{nm}$）与连续体（$<175\ \mathrm{nm}$）。
- **UVC**：$200$–$280\ \mathrm{nm}$（$5\times 10^4$–$3.57\times 10^4$）
   $\mathrm{O_3}$ **Hartley** 强吸收（峰值 $\sim 255\ \mathrm{nm}$），近乎不达地表；亦有 $\mathrm{O_2}$ **Herzberg** 带（$\sim 200$–$242\ \mathrm{nm}$）。
- **UVB**：$280$–$315\ \mathrm{nm}$（$3.57\times 10^4$–$3.17\times 10^4$）
   $\mathrm{O_3}$ **Huggins** 弱—中等吸收（$310$–$360\ \mathrm{nm}$ 向长波延伸），部分到达地表，主导多种光解反应。
- **UVA**：$315$–$400\ \mathrm{nm}$（$3.17\times 10^4$–$2.5\times 10^4$）
   大气吸收很弱，瑞利散射强；几乎可达地面。
   *备注*：$\mathrm{O_3}$ **Chappuis** 宽弱带横跨可见—近红外（$\sim 450$–$850\ \mathrm{nm}$），是背景校正时需考虑的缓慢吸收。

# 近红外（NIR）

**光谱学总范围**：$0.78$–$2.5\ \mu\mathrm{m}$（$1.28\times 10^4$–$4\times 10^3\ \mathrm{cm^{-1}}$）
 该段以**太阳反射**为主（热辐射贡献小），以振动**倍频与合频**、电子跃迁尾部为特征，气体线较弱但密集。
 **按大气吸收带/窗口细分（便于检索与检校）：**

- **NIR-1**：$0.78$–$1.00\ \mu\mathrm{m}$（$1.28\times 10^4$–$10^4$）
   关键：$\mathrm{O_2}$ **B** 带（$0.69\ \mu\mathrm{m}$）、**A** 带（$0.76\ \mu\mathrm{m}$）；$0.86$–$0.9\ \mu\mathrm{m}$ 为高透过窗口但邻近 $\mathrm{H_2O}$ $0.94\ \mu\mathrm{m}$ 吸收。
- **NIR-2**：$1.00$–$1.30\ \mu\mathrm{m}$（$10^4$–$7690$）
   窗口在 $\sim 1.03$ 与 $\sim 1.24\ \mu\mathrm{m}$；$\mathrm{H_2O}$ $1.13\ \mu\mathrm{m}$、$1.38\ \mu\mathrm{m}$ 两个强带夹出“**J**”波段窗口（$\sim 1.24\ \mu\mathrm{m}$）。
- **NIR-3**：$1.30$–$1.70\ \mu\mathrm{m}$（$7690$–$5880$）
   $1.55$–$1.65\ \mu\mathrm{m}$ 为重要窗口（“**H**”波段），用于 $\mathrm{CO_2}$（$1.6\ \mu\mathrm{m}$ 弱带）、$\mathrm{CH_4}$（$1.66\ \mu\mathrm{m}$）等检索；$1.4\ \mu\mathrm{m}$ 为强水汽带。
- **NIR-4**：$1.70$–$2.10\ \mu\mathrm{m}$（$5880$–$4760$）
   中间夹着 $\mathrm{H_2O}$ 强吸收 $1.9\ \mu\mathrm{m}$；$2.0$–$2.1\ \mu\mathrm{m}$ 窗口常用于地表/云与痕量气体。
- **NIR-5**：**$2.10$–$2.50\ \mu\mathrm{m}$**（$4760$–$4000$）
   遥感上常并入 **SWIR**；含 $\mathrm{CO_2}$ 强带（$\sim 2.06\ \mu\mathrm{m}$）与 $\mathrm{CH_4}$（$\sim 2.3\ \mu\mathrm{m}$），为柱浓度反演的主力段；$2.2\ \mu\mathrm{m}$ 附近是高透过窗口（“**K**”波段）。

# 中红外（MIR）

**光谱学总范围**：$2.5$–$25\ \mu\mathrm{m}$（$4000$–$400\ \mathrm{cm^{-1}}$）
 以**热辐射**和分子**基频振动**为主，选择定律强，气体吸收强烈。工程上常再分为 **MWIR** 与 **LWIR**。

- **MIR-A（MWIR）**：$3$–$5\ \mu\mathrm{m}$（$3333$–$2000$）
   *窗口*：$3$–$5\ \mu\mathrm{m}$ 为大气半窗口（既有地表/云热辐射也有太阳反射）。
   *主带*：$\mathrm{CH_4}\ 3.3\ \mu\mathrm{m}$，$\mathrm{CO_2}\ 4.3\ \mu\mathrm{m}$（$\nu_3$），$\mathrm{N_2O}\ 4.5\ \mu\mathrm{m}$，$\mathrm{CO}\ 4.67\ \mu\mathrm{m}$，$\mathrm{H_2O}$ 广泛带系。
- **MIR-B**：$5$–$8\ \mu\mathrm{m}$（$2000$–$1250$）
   多数气体强吸收：$\mathrm{SO_2}\ 7.3$–$8.7\ \mu\mathrm{m}$、$\mathrm{CH_4}\ 7.66\ \mu\mathrm{m}$（$\nu_4$）、$\mathrm{N_2O}\ 7.8\ \mu\mathrm{m}$、$\mathrm{NO_2}\ \sim 6.2\ \mu\mathrm{m}$、$\mathrm{H_2O}$ 密集带与连续体。
- **MIR-C（LWIR 窗口段）**：$8$–$12\ \mu\mathrm{m}$（$1250$–$833$）
   典型**大气窗口**，用于地表/云顶温度、地物热特性反演；中间被 $\mathrm{O_3}\ 9.6\ \mu\mathrm{m}$ 强带切开；$\mathrm{NH_3}$（$\sim 10.3\ \mu\mathrm{m}$）、$\mathrm{HNO_3}$（$\sim 11\ \mu\mathrm{m}$）等亦在此域。
- **MIR-D**：$12$–$17\ \mu\mathrm{m}$（$833$–$588$）
   $\mathrm{CO_2}\ 15\ \mu\mathrm{m}$（$\nu_2$）主带，温度廓线/温室效应诊断的核心通道；周围有 $\mathrm{H_2O}$ 旋转–振动带。
- **MIR-E（向远红外过渡）**：$17$–$25\ \mu\mathrm{m}$（$588$–$400$）
   以 $\mathrm{H_2O}$ 旋转带与连续体为主，辐射强度随温度显著变化，常用于对流层含水量与云冰相判识的补充。

## 实用备注（面向检索/反演）

- **气体优选波段**
  - $\mathrm{O_2}$：$0.69$、$0.76\ \mu\mathrm{m}$（A、B 带）；
  - $\mathrm{O_3}$：$0.255\ \mu\mathrm{m}$（Hartley）、$0.32$–$0.34\ \mu\mathrm{m}$（Huggins）、$9.6\ \mu\mathrm{m}$；
  - $\mathrm{H_2O}$：$0.94$、$1.13$、$1.38$、$1.9\ \mu\mathrm{m}$ 与 $>5\ \mu\mathrm{m}$ 多带系；
  - $\mathrm{CO_2}$：$1.6$、$2.06$、$4.3$、$15\ \mu\mathrm{m}$；
  - $\mathrm{CH_4}$：$1.66$、$2.3$、$3.3$、$7.66\ \mu\mathrm{m}$；
  - $\mathrm{CO}$：$2.3$（弱）、$4.67\ \mu\mathrm{m}$（强）；
  - $\mathrm{SO_2}$：$0.31$–$0.32\ \mu\mathrm{m}$、$7.3$–$8.7\ \mu\mathrm{m}$；
  - $\mathrm{NO_2}$：$0.4$–$0.45\ \mu\mathrm{m}$、$6.2\ \mu\mathrm{m}$。
- **窗口优先级**（地面/低轨成像）：
   $0.86$–$1.04$、$1.20$–$1.30$、$1.55$–$1.75$、$2.05$–$2.35\ \mu\mathrm{m}$（反射主导）；$3$–$5$ 与 $8$–$12\ \mu\mathrm{m}$（热辐射主导）。

若你有**具体任务的分界**（例如把 $2.1$–$2.5\ \mu\mathrm{m}$ 归到 SWIR、或把 MIR 只指 $3$–$5\ \mu\mathrm{m}$ 的 MWIR），告诉我区间即可，我按该口径给出对应清单与目标气体/窗口。

假设此时的信号光与选择的本征光信号为最简单情况下的单色光，其电场分量各自表示如下 ：

信号光：

$$e_s(t) = A_s \cos \omega_s t$$

本振光：

$$e_{lo}(t) = A_{lo} \cos \omega_{lo} t$$

当这两束光在耦合器束合后：

$$e(t) = A_s \cos \omega_s t + A_{lo} \cos \omega_{lo} t$$

并入射到 InGaAs 探测器表面，探测器的输出电流表示为：

$$i(t) = \alpha e^2(t) = \alpha (A_s \cos \omega_s t + A_{lo} \cos \omega_{lo} t)^2$$

经过简化处理后变为：

$$i(t) = \alpha (A_s^2 \frac{1+\cos 2\omega_s t}{2} + A_{lo}^2 \frac{1+\cos 2\omega_{lo} t}{2}) + \alpha A_s A_{lo} (\cos(\omega_s t + \omega_{lo} t) + \cos(\omega_s t - \omega_{lo} t))$$

式中 $e_s$ 和 $e_{lo}$ 分别表示太阳光和 LO 本征光的辐射相关电场的组成部分，$A_s$ 和 $A_{lo}$ 分别表示各自场的振幅，$\omega_s$ 是激光频率，$\omega_{lo}$ 是太阳光频率，$t$ 是在电场中的时间，$i$ 是由快速光电探测器测量的电流。

由于式中 $\cos 2\omega_s t$、$\cos 2\omega_{lo} t$、$\cos(\omega_s t + \omega_{lo} t)$ 属于高频率项，远大于探测器的响应带宽，因而 5.5 式可以简化为：

$$i(t) = \frac{\alpha (A_s^2 + A_{lo}^2)}{2} + \alpha A_s A_{lo} \cos(\omega_s t - \omega_{lo} t)$$

其中第一项为直流 DC 信号，第二项为交流 AC 信号。

直流 DC 信号：

$$i_{dc} = \frac{\alpha (A_s^2 + A_{lo}^2)}{2}$$

此式表示为此探测器对信号光（太阳光）和本振光（激光）的直流响应。

交流 AC 信号：

$$i_{ac} = \alpha A_s A_{lo} \cos((\omega_s - \omega_{lo})t)$$

交流信号经射频放大器及带通滤波器后由肖特二极管转化为 DC 信号，表示为：

$$I_m(t) \propto \beta A_s^2 A_{lo}^2 = \beta I_s I_{lo}$$

因此，外差信号探测的是太阳光强与本振光强的乘积，即激光外差是一个光放大的技术。

由于外差使用的光源为宽带太阳光，则总探测信号可以表示为：

$$I_m^{\alpha}(t) = \beta I_{lo}(t) \int_{v-\Delta v}^{v+\Delta v} I_s(v) dv$$

其中，$\Delta v$ 为射频检测带宽，$v(t)$ 是本振光在时间 $t$ 时的频率。当阳光穿过大气层时，地球上太阳光的光谱功率密度可以表示为：

$$I_s(v) = I \exp[-\alpha(v)]$$

其中 $I$ 假定为一个微小波长范围内的常数，是没有吸收的太阳光的光谱功率密度。而 $\alpha(v)$ 是沿太阳辐射路径的总吸收度。将式 5.11 带入式 5.10：

$$I_m^{\alpha}(t) = \beta I_{lo}(t) I \int_{v-\Delta v}^{v+\Delta v} \exp[-\alpha(v)] dv$$

本工作中使用的是逐点扫描模式，使激光器的工作电流逐点变化，同时用锁相放大器输出频率为 $f$ 的高频正弦波对激光器的注入电流进行调制，激光器的输出频率 $v(t)$ 和外差信号光强 $I_0(t)$ 表示为：

$$v(t) = v_0(t) + a \cos(2\pi ft)$$

$$I_0(t) = \bar{I_0} [1 + i_0 \cos(\omega t + \psi_1) + i_2 \cos(2\omega t + \psi_2)]$$

其中，$v_0$ 为所选激光器的实时工作波数 (单位 $\text{cm}^{-1}$)，$\bar{I_0}$ 为外差信号没有吸收时的平均光强，$a$ 为其调制振幅 (单位 $\text{cm}^{-1}$)，$i_0$ 是外差信号调制的相对强度，$\psi_1$ 是线性的 FM (频率调制) / IM (外差信号强度调制) 的相移。$\psi_2$ 是非线性的 FM/IM 的相移。

当激光外差信号到达探测器，此时透射光信号表示如下：

$$I(t) = I_0(t) \tau(v + a \cos \omega t)$$

当满足 $\alpha(v) \ll 0.1$ 弱吸收条件，利用傅里叶余弦级数展开：

$$\tau(\bar{v} + a \cos \omega t) \approx 1 - \alpha(\bar{v} + a \cos \omega t) = - \sum_{k=0}^{\infty} H_k(\bar{v}, a) \cos(k \omega t)$$

当 $\alpha(v) = PX_{gas} S_i(T) \phi L$，得到 K 阶傅里叶系数

$$H_0(\bar{v}, a) = - \frac{PSXL}{\pi} \int_{-\pi}^{\pi} \phi(\bar{v} + a \cos \theta) d\theta$$

$$H_k(\bar{v}, a) = - \frac{PSXL}{\pi} \int_{-\pi}^{\pi} \phi(\bar{v} + a \cos \theta) \cos k\theta d\theta, \ k = 1, 2, 3...$$

由于在波长调制技术中，$2f$ 信号具有波形对称、峰值位于吸收线、中心信号最强等优点，因此在实际工作中，利用锁相放大器对于外差信号进行解调，以此获得 $X_{2f}$ 和 $Y_{2f}$ 分量。

$$X_{2f} = \frac{G \bar{I_0}}{2} [H_2 + \frac{i_0}{2}(H_1+H_3)\cos\psi_1 + i_2(1+H_0 + \frac{H_4}{2})\cos\psi_2]$$

$$Y_{2f} = -\frac{G \bar{I_0}}{2} [i_0(H_1-H_3)\sin\psi_1 + i_2(1+H_0 - \frac{H_4}{2})\sin\psi_2]$$

无吸收时：

$$X_{2f}^0 = \frac{G \bar{I_0}}{2} i_2 \cos\psi_2$$

$$Y_{2f}^0 = \frac{G \bar{I_0}}{2} i_2 \sin\psi_2$$

去除无吸收，此时二次谐波信号为

$$S_{2f} = \frac{G \bar{I_0}}{2} \left\{ \begin{array}{l} [H_2 + \frac{i_0}{2}(H_1+H_3)\cos\psi_1 + i_2(H_0 + \frac{H_4}{2})\cos\psi_2]^2 \\ + [i_0(H_1-H_3)\sin\psi_1 + i_2(H_0 - \frac{H_4}{2})\sin\psi_2]^2 \end{array} \right\}$$
